America Hikes

Link to the Final Project. A Web Application in MEAN STACK that targets users who are interested in Trekking and Trails. The application searches for a trail/trek filtered by state, displays the location of the trail/trek and provides information about possible climatic conditions for an input date. The website contains Information about hundreds of treks and trails spread across America.
